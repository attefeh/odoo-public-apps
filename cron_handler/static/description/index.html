Cron Handler

Flexible schedules for any Odoo model method

Cron Handler lets you run methods on any concrete Odoo model on daily, weekly, or monthly timetables — using local time and a per-handler timezone. No need to maintain dozens of separate scheduled actions.

Key features

  • Multiple schedule lines per handler
  • Per-handler timezone (company default)
  • Link existing ir.cron to pre-fill model/method
  • Duplicate protection per hour slot
  • User / Manager security groups
  • Company-aware records
  • Works on Odoo 18 Community & Enterprise
  • No extra Python dependencies

Configuration

After installation, go to Settings → Technical → Cron Handlers, create a handler, add schedule lines, and implement your target method (e.g. cron_handler_run) on the chosen model.
