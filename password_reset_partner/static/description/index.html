🔐 Reset User's Password

This module allows administrators to securely reset passwords for user accounts, including Portal users and Internal Users from their partner's form.

✨ Key Features

  • Reset passwords directly from the partner form
  • Audit log of password reset actions
  • Compatible with both community version and enterprise version
  • Compatible with multicompany database

✅ Compatibility

  • Odoo versions: 18.0 and 19.0
  • Editions: Community and Enterprise
  • Hosting: Odoo.sh, On-Premise and Odoo Online (SaaS)

⚙️ Technical Notes

This module adds a button to the res.partner form view.
When clicked, it opens a wizard to ask for new password. By confirming, it will update related user to that partner's password.

📦 Installation

  • Copy the module to your Odoo addons directory
  • Update the app list
  • Install the module from the Apps menu
