🔐 Reset User's Password

This module allows administrators to securely reset passwords for user accounts — including Portal and Internal users — directly from the partner form.

✨ Key Features

  • Reset passwords directly from the partner form.
  • Audit log of password reset actions.
  • Compatible with both Community and Enterprise editions.
  • Compatible with multi-company databases.

⚙️ Technical Notes

This module adds a button to the res.partner form view. When clicked, it opens a wizard asking for a new password; on confirmation, the related user's password is updated.

✅ Compatibility

  • Odoo versions: 18.0 and 19.0
  • Editions: Community and Enterprise
  • Hosting: Odoo.sh, On-Premise and Odoo Online (SaaS)

📦 Installation

  • Copy the module to your Odoo addons directory.
  • Update the app list.
  • Install the module from the Apps menu.
